## Mac OS X with Homebrew
Open a terminal, and then run the following commands:

```
brew update
brew upgrade dvm
```

## Linux and Mac OS X without Homebrew
Open a terminal, and then run the following command:

```
dvm upgrade
```

## Windows
Open a terminal, and then run the following command:

```
dvm upgrade
```
